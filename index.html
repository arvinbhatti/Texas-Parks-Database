The Goal

Our Texas Parks Database is a web app that will help its users plan their next visit to a Texas park.

Our Vision

The idea is that our application will provide a quick and useful overview of all the information that is necessary for a potential visitor to make the most informed decision about what park is most ideal for their next visit.

Technologies

We will use the Google Geocoding and Maps API to plot the parks location on a map. We will also use this API to find the location of the parks in order to get the weather information from the Dark Sky API and OpenWeatherMap.
